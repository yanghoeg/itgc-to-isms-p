# 04-laws

ISMS-P 인증기준과 직접 연결되는 한국 법령 정리.

## 핵심 법령

- **「개인정보 보호법」** — ISMS-P 3장 전체의 기반 법령
- **「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 (정통망법)** — 2장·3장 일부
- **「전자금융거래법」 / 「전자금융감독규정」** — 금융권 한정. 비금융권 비교 참조용
- **「신용정보의 이용 및 보호에 관한 법률」 (신용정보법)** — 신용정보 처리 관련 영역

## 작성 원칙

- 조문 번호와 제목은 인용 가능
- 조문 본문 verbatim 인용 금지 — 작성자의 말로 요약
- 원문 링크는 국가법령정보센터(https://law.go.kr)로 통일
- 개정 이력 추적 — 최신 시행일 기준으로 정리
