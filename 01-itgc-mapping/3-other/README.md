# 3-other — ITGC 거의 무관, 참고 가치 있는 영역

회계감사 ITGC가 직접·간접 모두 다루지 않는 영역. 다만 ISMS-P 인증을 받으려는 클라이언트 관점에서는 동등하게 중요하며, ITGC 감사인이 클라이언트 통합 view에서 참고할 수 있다.

| 부영역 | 폴더 | ISMS-P 매핑 | 개수 |
|---|---|---|---|
| **privacy-lifecycle** | [`privacy-lifecycle/`](./privacy-lifecycle/) | 3장 — 개인정보 처리 단계별 (수집·이용·제공·파기·권리) | 21 |
| **합계** | | | **21** |

> 회계감사 IT 감사인이 가장 약한 영역. ISMS-P 응시생 입장에서는 별도 학습 노력이 가장 크게 필요한 부분.
