# ISMS-P × ITGC 매핑 매트릭스

KISA 2023.11 안내서 기준 101개 인증기준 양방향 인덱스. ISMS 번호 ↔ ITGC 영역 1:1 또는 1:다 매핑.

> 인증기준 명칭은 작성자 기억 기반. KISA 안내서 원문과 정확히 일치하지 않을 수 있음. 페이지 작성 시 정확한 표기로 점진 보정.
> 페이지는 점진 작성. `→` 표기는 작성 완료, `(작성 예정)` 표기는 미작성.
