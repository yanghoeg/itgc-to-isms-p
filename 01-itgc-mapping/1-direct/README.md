# 1-direct — ITGC 직접 매핑 영역

ITGC 표준 4영역(APD/PC/CO/PD)에 직접 매핑되는 ISMS-P 인증기준. 회계법인 IT 감사인이 ITGC 감사 절차로 자연스럽게 검증하는 영역이다.

| 약어 | 영역 | 폴더 | ISMS-P 매핑 | 개수 |
|---|---|---|---|---|
| **APD** | Access to Programs and Data | [`apd/`](./apd/) | 2.5, 2.6, 2.7.2(키관리) | 14 |
| **PC** | Program Changes | [`pc/`](./pc/) | 2.8.6(운영환경 이관), 2.9.1(변경관리) | 2 |
| **CO** | Computer Operations | [`co/`](./co/) | 2.4(물리), 2.9, 2.10, 2.11, 2.12 | 28 |
| **PD** | Program Development | [`pd/`](./pd/) | 2.8.1~2.8.5 | 5 |
| **합계** | | | | **49** |

> CO는 항목 수가 많아 5개 부영역(operations / security / incident / dr / physical)으로 세분화.
