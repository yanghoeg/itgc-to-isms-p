# 01-itgc-mapping

ITGC 4대 영역을 ISMS-P 인증기준에 매핑한 메인 콘텐츠.

| 영역 | 주요 ISMS-P 인증기준 | 폴더 |
|---|---|---|
| Access to Programs and Data | 2.5 인증 및 권한 관리, 2.6 접근 통제 | [`access-to-data/`](./access-to-data/) |
| Change Management | 2.8 정보시스템 도입 및 개발 보안, 2.9.5 변경 관리 | [`change-management/`](./change-management/) |
| Computer Operations | 2.9 시스템 및 서비스 운영, 2.11 사고 예방·대응, 2.12 재해복구 | [`computer-operations/`](./computer-operations/) |
| Program Development | 2.8 정보시스템 도입 및 개발 보안 | [`program-development/`](./program-development/) |

각 영역 폴더 안에는 ISMS-P 인증기준 번호별 매핑 페이지가 들어간다 (예: `2.5.1-사용자-계정-관리.md`). 페이지 구성은 [`../00-meta/methodology.md`](../00-meta/methodology.md) §2 참조.

> 작성 진행 중. 완료된 인증기준은 일부이며, 우선순위는 [`../00-meta/methodology.md`](../00-meta/methodology.md) §3 참조.
