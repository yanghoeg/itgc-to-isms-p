# 03-isms-p-only

ITGC와 직접 매핑되지 않는 ISMS-P 고유 영역. 회계감사 ITGC 절차에서는 잘 다루지 않지만, ISMS-P 인증을 받으려는 클라이언트 관점에서는 동등하게 중요한 영역이다.

## 주요 영역

- **2.4 물리 보안** — 출입통제, 정보시스템 보호구역, 보호구역 작업
- **2.7 암호화 적용** — 암호정책 수립, 암호 키 관리
- **3장 개인정보 처리단계별 요구사항 (21개 인증기준)** — 수집, 이용·제공, 보유·파기 단계별 통제

회계감사 ITGC 백그라운드에서 가장 약한 영역이며, ISMS-P 응시생 입장에서 별도 학습 노력이 가장 크게 필요한 영역이다.
